<h1">버티의 머지 베이커리 게임의 개인정보 처리방침.</h1>
<p>버티의 머지 베이커리는 아무런 권한을 요구하지 않으며, 개인정보를처리하지 않고, 개인을 식별 할 수 있는 정보를 저장하지 않습니다.</p>
